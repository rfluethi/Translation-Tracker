Zeigt den Übersetzungsfortschritt von learn.wordpress.org als Dashboard auf deiner WordPress-Seite.

== Description ==

Das Plugin liest Übersetzungs-Issues aus dem GitHub Repository WordPress/Learn
und zeigt den Fortschritt als interaktives Dashboard an.

**Features:**

* Shortcode `[translation_tracker]` für jede Seite/Beitrag
* Liest Status-Tabellen direkt aus GitHub Issues
* Serverseitiges Caching (konfigurierbar)
* Filter, Suche und Fortschrittsbalken
* CSV-Export
* Aktualisieren-Button (AJAX, ohne Seitenreload)
* Passt sich an das aktive WordPress-Theme an (Light Design)
* Einstellungsseite unter Einstellungen → Translation Tracker

== Installation ==

1. Plugin-Ordner `wp-translation-tracker` nach `wp-content/plugins/` hochladen
2. Plugin im WordPress-Admin aktivieren
3. Unter Einstellungen → Translation Tracker konfigurieren
4. Shortcode `[translation_tracker]` auf einer Seite einfügen
